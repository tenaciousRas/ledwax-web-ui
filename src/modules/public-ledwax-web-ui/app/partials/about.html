A brief overview of the setup for this application:

  • Setup a particle cloud (or register your user on the public cloud).
  • Register the Particle Photon IoT devices on the Particle cloud.
  • Flash Photon devices with the LEDWax firmware.
  • Setup this application (middleware API) with the IP address and client credentials of the cloud host.

LEDWax is intended primarily for a private Particle cloud but can also function with the public Particle cloud. To configure cloud hosts add records to the LEDWax API (middleware) database. Currently this must be done manually.

To use the application:

  • To Login use the username/password already created on your Particle cloud server.
  • Setup and store LEDWax devices on the 'Setup' screen. Devices only need to be setup once.
  • Once LEDWax devices are stored, they can be driven from the LEDs screen. You can set mode, color, brightness, fade mode, and more.
